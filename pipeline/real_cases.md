# real_cases.md — จุดเจ็บ/เคสจริงของคนไทย (poor-man's RAG ป้อนสภา)
# แต่ละบรรทัด = สถานการณ์จริงที่เจอบ่อย ใช้เป็น context ให้คอนเทนต์ "จี้จุด" ไม่ใช่ภาษา AI ลอยๆ
# (เพิ่ม/แก้ได้เรื่อยๆ จากกระทู้ Pantip/คอมเมนต์จริง)
หนี้บัตรหลายใบ จ่ายขั้นต่ำทุกเดือนแต่ยอดไม่ลด ดอกทบจนท้อ
เงินเดือน 15000-18000 อยากได้บัตรใบแรกแต่กลัวสมัครไม่ผ่าน ไม่รู้เริ่มจากใบไหน
ฟรีแลนซ์/ค้าขาย รายได้ไม่ประจำ ยื่นบัตร/สินเชื่อยากเพราะไม่มีสลิปเงินเดือน
สมัครบัตรหลายใบพร้อมกันแล้วโดนปฏิเสธ ไม่รู้ว่ายื่นถี่มีผลต่อเครดิตบูโร
อยากปิดหนี้แต่ไม่รู้จะเริ่มก้อนไหนก่อน (ดอกสูงสุด vs ยอดน้อยสุด)
จะรีไฟแนนซ์บ้าน/รถ แต่ไม่รู้คุ้มไหม คิดค่าใช้จ่าย/ค่าธรรมเนียมยังไง
อยากมีเงินสำรองฉุกเฉินแต่เก็บไม่อยู่ พอมีเรื่องด่วนก็หมด
โดนทวงหนี้ เครียด อยากเข้าโครงการไกล่เกลี่ย/คลินิกแก้หนี้ แต่ไม่รู้ขั้นตอน/เอกสาร
ต้องการเงินด่วน คิดจำนำทะเบียนรถ กลัวดอกแพง/โดนยึด
มือใหม่ทำงานปีแรก อยากเริ่มวางแผนการเงิน/ออม แต่ข้อมูลเยอะจนงง
อยากได้บัตรสะสมไมล์/cashback แต่ไม่รู้ใบไหนคุ้มกับไลฟ์สไตล์ตัวเอง
ผ่อนรถอยู่แต่เริ่มผ่อนไม่ไหว ควรขายดาวน์/รีไฟแนนซ์/คืนรถ
