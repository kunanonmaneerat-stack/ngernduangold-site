# 🔄 ระบบวงจรปิด ngernduangold — แผนผังรวม (handoff ได้ทั้งหมด)

Cowork คุมหัว → กระจายงานผ่าน agent หลายตัว → ตรวจ 5 ชั้น → ส่ง Claude Code execute →
รับผลกลับ → Cowork วิเคราะห์ยกระดับ → ป้อน dispatcher วนรอบใหม่ **โดยไม่หลุด compliance**

## วงจร (ใคร/ไฟล์/คำสั่ง)

```
①  Cowork/เจ้าของ ตั้งออเดอร์            → automation-log/orders.txt
②  dispatcher.py   กระจายงาน            → ทีละหัวข้อเข้าสภา
③  content_council.py  สภา 5 agent      → council-<date>.md  (draft ตรวจแล้ว)
        expert · comply · value · review · correct   (สมอง = free_llm: GLM→Qwen)
④  executor.py     รวมแผน + ส่ง CC      → cc-inbox/order-*.md   (+ping Telegram)
⑤  Claude Code     execute แผน          → cc-outbox/result-*.md   [CC = ตัวหลัก]
        (สำรองเมื่อ CC ไม่ว่าง: cc_runner.py ใช้ free_llm)
⑤·5 comply_gate.py ตรวจ compliance ซ้ำ   → ตัด trigger/เติม caveat (auto-fix) + stamp PASS/⚠️
⑥  report_to_cowork.py  รับผล CC        → cowork-review/review-*.md  (+ping, โชว์ verdict)
⑦  Cowork วิเคราะห์ ยกระดับ/ปรับปรุง     → เขียน orders.txt รอบใหม่ → กลับ ②
```

## คำสั่งเดียวจบครึ่งวง
```
py pipeline\loop.py plan      # ② ③ ④  (กระจาย→ตรวจ→ส่ง CC)
py pipeline\loop.py collect   # ⑥      (เก็บผล CC→ให้ Cowork)
```
รันอัตโนมัติ: Windows task `ngernduangold-daily-queue` (ทุก 07:30) → `pipeline\run_daily.cmd`

## agent ครบชุด
| # | agent | ไฟล์ | หน้าที่ |
|---|---|---|---|
| สมอง | free_llm | free_llm.py | พูลโมเดลฟรี GLM→Qwen→… สลับเมื่อเต็ม (คีย์จาก registry) |
| ตรวจ | สภา 5 | content_council.py | expert/comply/value/review/correct |
| รับออเดอร์+กระจาย | dispatcher | dispatcher.py | orders.txt → วนเข้าสภา → คิว → ping |
| รวม+ส่ง CC | executor | executor.py | คิว → work order → cc-inbox |
| สะพาน CC | cc_bridge | cc_bridge.py | send()/collect() + ping Telegram |
| execute สำรอง | cc_runner | cc_runner.py | ทำแทนเมื่อ CC ไม่ว่าง (มี gate ในตัว) |
| ตรวจหลัง CC | comply_gate | comply_gate.py | สแกน trigger/caveat + auto-fix + stamp |
| รับผล→Cowork | report_to_cowork | report_to_cowork.py | cc-outbox → cowork-review (โชว์ verdict) |
| ออร์เคสเตรเตอร์ | loop | loop.py | สั่งทีละ stage |

ลิงก์อยู่ DM/bio เท่านั้น · งานแตะของจริง = ใส่ "ธงต้องขออนุมัติ" ให้เจ้าของกด

## ⚠️ บทเรียนรอบแรก (ทำไมต้องมี Cowork คั่น)
ตัว execute (CC/cc_runner) **regress ได้** — รอบเทสต์ GLM ใส่คำ "ฟรี" + ฟันธง "0%" ทิ้ง caveat
→ **แก้แล้ว:** `comply_gate.py` คั่นหลัง CC อัตโนมัติ — สแกนคำ trigger + ตัวเลข %/ดอกเบี้ยต้องมี caveat,
