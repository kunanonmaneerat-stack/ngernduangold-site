## เสียงตรงกัน (ทำเลย — มั่นใจสูง)
- Pantip/Threads = value ลึกก่อน แล้วค่อยชวน DM/bio · ไม่มีลิงก์ในเนื้อหา
- IG/TikTok = **comment-keyword → auto-DM** (ManyChat: คอมเมนต์ "CHECK"/"เช็กหนี้" → ระบบส่งลิงก์ Quiz) = ปั๊ม lead เลี่ยง throttle ลิงก์
- FB = **Poll → DM รายคน** คัด intent ก่อนยิงลิงก์
- ทุกแพลตฟอร์ม = แนบคำเตือน **Responsible Lending ธปท.** "กู้เท่าที่จำเป็น ชำระคืนตามกำหนด"

## Opus ยกระดับเพิ่มจาก Gemini (ของใหม่)
1. **DM pre-qualify 2 คำถามก่อนส่งลิงก์** — auto-DM ของ Gemini ดี แต่เติม: ถาม "มีหนี้/บัตรกี่ใบ?" + "รายได้ต่อเดือน?" ก่อนส่ง quiz → ลด bounce เพิ่ม % ทำ quiz จบ
2. **รีเฟรม Threads hook ให้ปลอดภัยขึ้น** — "ความลับสถาบันการเงินที่ไม่อยากให้คุณรู้" เสี่ยงดูแฉ/ชี้นำ → เปลี่ยนเป็น **"3 อย่างที่ทำให้โปรไฟล์เราน่าอนุมัติขึ้น (ที่ไม่ค่อยมีใครบอก)"** — คงความอยากรู้ แต่ไม่โจมตีสถาบัน
3. **1 asset → 6 ฟอร์แมต (Quiz→Content loop)** — 1 คำถาม/ผลลัพธ์ของ /quiz = กระทู้ Pantip + carousel IG + hook TikTok + ตารางเทียบ YT + thread Threads + poll FB → ผลิตจากแหล่งเดียว (ประหยัดแรง + ข้อความสอดคล้องกัน)
4. **First-win re-route** — ช่วง pre-conversion ดันโปรดักต์ที่อนุมัติง่ายสุดก่อน (ประกันเดินทาง/บัตร lifestyle) เพื่อเก็บ conversion แรกให้ระบบ track ติด
5. **กรอบ "รวบรวมข้อมูล" ไม่ใช่ "ที่ปรึกษา"** (จาก Gemini) → ฝังเป็นกฎระบบเลย (อัป comply_gate.RULE)
