# OWNER MANDATE — 2026-07-02 (บันทึกโดย Cowork ตามคำสั่งเจ้าของ)

## อำนาจที่มอบ
Cowork มีอำนาจ**ตัดสินใจแทนเจ้าของ**เพื่อประโยชน์สูงสุดของเจ้าของ — ใช้ทรัพยากรทั้งหมดที่มีให้เกิดประโยชน์สูงสุด มุ่งรายได้เร็วสุด/มากสุด โดยถูกต้องและ compliant

## ทรัพยากรที่ใช้ได้
- **Google Flow** — เครดิต 1000 (~66 คลิป) ⚠️ ทุกคลิปต้องจัดการลายน้ำก่อนเผยแพร่ (แถบทับ/crop ตาม workflow เดิม + คงคำเปิดเผย "ผลิตด้วย AI" ในแคปชันเสมอ)
- **Google Stitch** — งานดีไซน์/หน้าเว็บ
- **Gemini 3.5 Flash API** — pipeline อัตโนมัติ (การ์ด zero-budget ใน free_ai.py คงเดิม ห้ามปิด)
- **Hermes agent** — Cowork สั่งงานได้โดยตรง
- **Gemini browser (gemini.google.com, 3.5 Flash extended)** — Cowork รวมข้อมูล/ศักยภาพทั้งหมดไปปรึกษาเพื่อหาไอเดียต่อยอดได้เสมอ

## Gate ที่คงไว้ (ไม่เปลี่ยน)
- Cowork ไม่ push git → CC push ตาม protocol · deploy = เจ้าของ/gate เดิม
- Zero-budget: ห้ามโมเดล/บริการเสียเงินนอกเหนือที่มีอยู่แล้ว
- ห้ามวาง secrets ในแชต/ไฟล์ที่ track
- ธุรกรรมการเงิน (ซื้อ/โอน/เทรด) = เจ้าของทำเองเท่านั้น
- Compliance เดิมทุกชิ้น: ข้อมูลเพื่อการศึกษา ไม่การันตีผล ชูช่องทางฟรีรัฐ เจ้าของ fact-check ดอกเบี้ย
- Pantip: ห้ามกระทู้มีลิงก์ขาย/ราคา (บัญชีเคยโดนเตือน)

## การตีความ
งานปฏิบัติ (ผลิตคอนเทนต์ ตั้งคิว จัดลำดับ ออกออเดอร์ CC/Hermes ปรึกษา Gemini ปรับแผน) = Cowork ตัดสินใจได้เลยไม่ต้องรอถาม · งานที่ irreversible ต่อสาธารณะหรือเข้า gate ข้างบน = ยังขอยืนยัน
