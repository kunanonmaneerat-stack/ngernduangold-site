## 📋 คลังคำตอบ Pantip (value-first · ไม่มีลิงก์ · ก๊อปแล้วปรับ)

**A) หนี้บัตรหลายใบ / จ่ายขั้นต่ำยอดไม่ลด**
> เคยเจอเหมือนกันครับ จ่ายขั้นต่ำทุกเดือนแต่ยอดแทบไม่ลด เพราะดอกบัตรสูง (ราว 16–25%/ปี) เลยโดนดอกกินเกือบหมด. 2 ทางที่คนส่วนใหญ่ใช้: (1) ลิสต์หนี้ทุกใบ + ดอกแต่ละใบ แล้วโปะใบดอกแพงสุดก่อน (2) ถ้าหลายใบจริง ลองดูสินเชื่อรวมหนี้ดอกต่ำกว่าบัตร เอามาปิดบัตรทีเดียวแล้วผ่อนก้อนเดียว — แต่ต้องเทียบดอก+ค่าธรรมเนียมให้ดีก่อน และห้ามรูดเพิ่มระหว่างผ่อน. ถ้าอยากได้ตัวเทียบสินเชื่อรวมหนี้แบบกดเทียบดอกง่าย ๆ บอกได้ เดี๋ยวแปะให้ทาง DM / อยู่ในโปรไฟล์ครับ

**B) เงินเดือนน้อย สมัครบัตร/สินเชื่อไม่ผ่าน**
> เงินเดือน 15,000 สมัครบัตรได้ครับ หลายธนาคารเริ่มเกณฑ์ที่ 15,000/เดือน. จุดที่คนพลาดบ่อย = สมัครหลายใบพร้อมกันในเดือนเดียว → ธนาคารเห็น hard inquiry ถี่ → ถูกปฏิเสธยกแผง. ทริค: เลือกใบที่เกณฑ์ตรงเงินเดือนเรา สมัครทีละใบเว้นระยะ, เตรียมสลิป/statement, เดินบัญชีสม่ำเสมอ. ถ้าอยากได้ลิสต์บัตรที่เริ่มเกณฑ์เงินเดือนไม่สูง + ค่าธรรมเนียม เดี๋ยวส่งให้ทาง DM / อยู่ในโปรไฟล์ครับ

**C) ออมเงินไม่อยู่ / เก็บไม่ได้**
> ที่เวิร์กกับมนุษย์เงินเดือน = แยกบัญชี "ออม" ออกจาก "ใช้จ่าย" แล้วโอนเข้าออมทันทีที่เงินเดือนเข้า ~10% ก่อนใช้ (เก็บก่อนจ่าย ไม่ใช่เหลือค่อยเก็บ). บัญชี/แอปออมดอกสูงตอนนี้ให้ดอกกว่าออมทรัพย์ปกติหลายเท่า ถอนได้ไม่ล็อก เหมาะเก็บเงินสำรองฉุกเฉิน (ควรมี 3–6 เท่าของรายจ่าย/เดือน). ถ้าอยากได้ตัวเทียบบัญชีออมดอกสูงถอนได้ บอกได้ครับ เดี๋ยวแปะใน DM/โปรไฟล์

**D) ต้องการเงินด่วน / จำนำทะเบียนรถ**
> ถ้าต้องการเงินด่วนก้อนนึงและมีรถ ลองดูจำนำทะเบียนรถครับ ดอกมักต่ำกว่ากู้นอกระบบมาก (นอกระบบบางทีเกิน 20%/เดือน) และรถยังขับได้ปกติ — แต่ดอกเริ่มต้นแล้วแต่ที่ ต้องอ่านเงื่อนไข + คำนวณดอกรวมก่อนเซ็น อย่าดูแค่ค่างวด. ถ้าอยากได้ตัวเทียบจำนำทะเบียน/สินเชื่อรถดอกต่ำไว้กดเทียบ บอกได้ เดี๋ยวส่ง DM / อยู่ในโปรไฟล์ครับ

> ⚠️ Pantip มองลิงก์ในคำตอบเป็นโฆษณา — ดังนั้น **ไม่แปะลิงก์ในคำตอบ** ใช้ "ทักมา/ดูโปรไฟล์" เป็นสะพานแทน

## 🤖 สั่ง Hermes รัน Daily Intent Engine (ก๊อปวางในแอป/Telegram ทุกเช้า)
> Hermes: รัน daily intent engine — (1) หากระทู้ Pantip + โพสต์ Threads/FB intent สูงวันนี้ หัวข้อ หนี้/สินเชื่อ/ออม/ประกัน คัด top 6 (2) ร่าง value-first answer ต่ออัน คปภ.-safe **ไม่มีลิงก์** (3) GA4 digest (filter 127.0.0.1) + AccessTrade clicks + บอก quiz_start สะสม. ทุกอย่างเข้าคิวรอ approve **ห้ามโพสต์เอง**

→ Hermes เตรียมให้ 80% เหลือคุณแค่ **approve + โพสต์เอง** 20% (= กัน shadowban). ผมสั่ง Hermes/CC แทนจาก session นี้ไม่ได้ — รันบนเครื่องคุณ.

## 🗓️ จังหวะ 2 อาทิตย์ (ตาม playbook)
- **W1:** รัน engine ทุกเช้า + Pantip 3–5/วัน + FB Groups + Threads → **สะสม quiz_start** (ยังไม่แตะโค้ด)
- **W2:** ถ้า `quiz_start ≥ 30` → บอกผม (หรือสั่ง CC) แก้ funnel **จุดเดียวที่ drop หนักสุด** · **ตัดช่องที่ quiz_start = 0** · จับ affiliate_click ตัวแรก

## 🔍 เช็คครั้งเดียว: AccessTrade
ดูรายงาน 30 วัน — มีคลิก (~33) + sub_id (`threads_`/`quiz_`/`links_`) ขึ้นไหม.
- คลิกมี แต่ conversion 0 = **ปกติ** (คนน้อย + อนุมัติยาก) → แก้ด้วยการเพิ่มคน = โพสต์
- คลิก 0 ทั้งที่ Google เห็น 33 = tracking ไม่ส่ง sub → **บอกผม เดี๋ยวไล่แก้ `build_site.py` ให้**
